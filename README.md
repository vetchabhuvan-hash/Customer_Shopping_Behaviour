# Customer_Shopping_Behaviour
This project analyzes customer shopping behavior to identify buying patterns, preferences, and trends. Using data analytics and visualization tools, it uncovers insights that help businesses personalize marketing, improve customer experience, and make data-driven decisions to boost sales and loyalty.
Overview

Process

Data Cleaning (Python):

Cleaned and prepared the raw dataset using Python (Pandas, NumPy).

Handled missing values, duplicates, and formatted data for database import.

Database Management (SQL):

Imported the cleaned dataset into SQL.

Created multiple tables and performed SQL queries for structured analysis.

Data Visualization (Power BI):

Imported SQL tables into Power BI.

Built interactive visuals and reports.

Created DAX measures to calculate KPIs like total sales, average order value, and customer frequency.

Tools & Technologies

Python: Pandas, NumPy

SQL: MySQL / SQL Server

Power BI: DAX, Power Query

Visualization: Charts, Slicers, KPIs

Key Insights

Identified top-selling products and high-value customers.

Discovered purchase patterns across demographics and time periods.

Highlighted areas for improving marketing and customer retention.

Outcome

An interactive dashboard that transforms raw customer data into meaningful insights, enabling smarter business decisions and performance tracking.
